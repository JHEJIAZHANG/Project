# 校園生活管理系統

一個專為學生設計的校園生活管理網站，整合了 Google 日曆、Google Classroom、筆記管理和待辦事項等功能，幫助學生更好地組織學習生活。

## ✨ 主要功能

### 🔑 核心功能

#### 📚 Google Classroom 整合
- **課程同步**：自動同步 Google Classroom 課程資料
- **作業管理**：顯示課程作業、截止日期和完成狀態
- **課程資訊**：查看課程詳情、教師資訊和學生數量
- **快速連結**：直接開啟 Google Classroom 課程頁面

#### 📅 Google 日曆整合
- **OAuth 2.0 認證**：安全的 Google 帳號登入
- **多視圖支援**：日、週、月視圖切換
- **事件同步**：即時同步 Google 日曆事件
- **狀態更新**：在頁面上直接更新事件狀態

#### 📝 筆記功能
- **分類管理**：課程、個人、工作分類
- **富文字編輯**：支援基本格式、標題、清單
- **標籤系統**：顏色標籤分類和管理
- **搜尋功能**：快速找到相關筆記

#### ✅ 待辦事項管理
- **CRUD 操作**：新增、編輯、刪除待辦事項
- **拖曳排序**：直覺的優先級調整
- **截止日期**：與日曆事件關聯
- **完成追蹤**：進度統計和視覺化

### 🎯 特色功能

#### 🔔 智慧提醒
- **瀏覽器通知**：事件和待辦到期提醒
- **即將到期**：自動識別緊急事項
- **自訂提醒**：個人化通知設定

#### 🎨 顏色管理
- **標籤系統**：區分不同事件類型
- **視覺化**：直覺的顏色編碼
- **自訂配色**：個人化主題設定

#### 📊 學習追蹤
- **學期進度條**：顯示目前學習進度
- **統計儀表板**：今日行程、待辦數量
- **課程概覽**：課程數量、學生數、作業統計

## 🎨 介面設計

### 技術架構
- **前端框架**：React 18 + Vite
- **樣式系統**：TailwindCSS + shadcn/ui
- **圖標庫**：Lucide React
- **狀態管理**：React Context API

### 設計特色
- **響應式設計**：完美支援桌機與手機
- **校園風格**：清新簡潔的學生友善介面
- **卡片佈局**：資訊層次分明
- **直覺導航**：側邊欄 + 頂部操作列

## 🚀 快速開始

### 環境需求
- Node.js 18+
- pnpm 或 npm

### 安裝步驟

1. **克隆專案**
```bash
git clone <repository-url>
cd campus-life-manager
```

2. **安裝依賴**
```bash
pnpm install
```

3. **啟動開發伺服器**
```bash
pnpm run dev
```

4. **開啟瀏覽器**
```
http://localhost:5173
```

### 建置部署

```bash
# 建立生產版本
pnpm run build

# 預覽生產版本
pnpm run preview
```

## 🛠️ 開發指南

### 專案結構
```
src/
├── components/          # 共用組件
│   ├── ui/             # UI 基礎組件
│   └── Layout.jsx      # 主要佈局
├── pages/              # 頁面組件
│   ├── Dashboard.jsx   # 儀表板
│   ├── Calendar.jsx    # 行事曆
│   ├── Courses.jsx     # 課程管理
│   ├── Notes.jsx       # 筆記
│   └── TodoList.jsx    # 待辦事項
├── contexts/           # 狀態管理
│   └── AppContext.jsx  # 應用程式狀態
└── App.jsx            # 主要應用程式
```

### 新增功能
1. 在對應的頁面組件中實現 UI
2. 在 AppContext 中加入狀態管理
3. 更新 Layout 組件的導航
4. 在 App.jsx 中加入路由

### API 整合準備
- 所有 Google API 呼叫都已模組化
- 預留後端 API 接口
- 錯誤處理和載入狀態
